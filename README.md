HERE SDK for Flutter — Navigate Edition Demo

A Flutter application built on the HERE SDK (Navigate Edition) demonstrating map display, POI/address search, route calculation, and real-GPS turn-by-turn navigation — structured with Clean Architecture and BLoC.

Features
Map display — live map centered on the device's current location, with a location marker
Search — address search
Routing — car route calculation with distance
Turn-by-turn navigation — real-GPS guided navigation with live maneuver instructions and distance-to-next-turn
